# doc-search Agent

Search and explore documents using semantic search.

## Capabilities

- **search**: Find relevant content across all indexed documents
- **read**: Read full document content
- **expand**: Get more context around a search result
- **list**: See all available documents
- **navigate**: Move through a document chunk by chunk

## Usage

Use the doc-memory MCP tools to search and explore documents.
Start with `search` to find relevant content, then use `expand` or `read` for more context.
